# PongInVHDL
Making classic game Pong in VHDL

# Features
- Classic Mode
- Enhanced AI Mode with Dynamic Difficulties
- Frenzy Mode
  - Allows moving horizontally
  - Special effect items can be picked up

# Hardware
- Zedboard
- VGA Display
- Pmod JSTK2: Two-axis Joystick
- Vibration Motor Module
